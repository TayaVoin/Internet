## Критические проблемы

### A01: Нарушение контроля доступа
- **Проблема**: Нет проверки владельца заказа в `/orders/{id}`
- **Последствия**: Любой пользователь может смотреть чужие заказы
- **Решение**: Добавить проверку
```python
if order.user_id != current_user.id:
    raise HTTPException(403, "Доступ запрещен")
```
