## Чек-лист (OWASP Top 10)

### A01:2021 – Broken Access Control
- [ ] Проверка: Можно ли получить заказ другого пользователя, изменив ID в URL? (`GET /orders/{id}`)
- [ ] Проверка: Можно ли удалить/изменить чужой заказ?
- [ ] Проверка: Есть ли ограничения по ролям (admin/user)?

### A02:2021 – Cryptographic Failures
- [ ] Проверка: Используется ли HTTPS (не HTTP)?
- [ ] Проверка: Передаются ли пароли/токены в открытом виде?
- [ ] Проверка: Хэшируются ли пароли (bcrypt/argon2, а не MD5)?
- [ ] Проверка: Не зашиты ли секреты в коде?

### A03:2021 – Injection
- [ ] Проверка: Есть ли уязвимости к SQL-инъекциям (если используется БД)?
- [ ] Проверка: Экранируются ли вводимые пользователем данные?
- [ ] Проверка: Используется ли ORM (SQLAlchemy, Prisma) вместо сырых запросов?

### A04:2021 – Insecure Design
- [ ] Проверка: Есть ли защита от brute-force на логин/регистрацию?
- [ ] Проверка: Валидируются ли все входящие данные (Pydantic/schemas)?

### A05:2021 – Security Misconfiguration
- [ ] Проверка: Отключен ли debug mode в продакшене?
- [ ] Проверка: Нет ли стандартных/слабых паролей?
- [ ] Проверка: Закрыты ли лишние порты в Docker контейнере?

### A06:2021 – Vulnerable and Outdated Components
- [ ] Проверка: Актуальные ли версии зависимостей (fastapi, uvicorn)?
- [ ] Проверка: Есть ли известные уязвимости в используемых библиотеках?

### A07:2021 – Identification and Authentication Failures
- [ ] Проверка: Используется ли JWT с коротким сроком жизни?
- [ ] Проверка: Проверяется ли подпись токена?
- [ ] Проверка: Есть ли механизм logout/revoke токенов?

### A08:2021 – Software and Data Integrity Failures
- [ ] Проверка: Подписываются ли обновления?
- [ ] Проверка: Не используется ли unsafe десериализация (pickle, yaml.load)?

### A09:2021 – Security Logging and Monitoring Failures
- [ ] Проверка: Логируются ли подозрительные действия (много 401/403)?
- [ ] Проверка: Есть ли мониторинг на необычную активность?

### A10:2021 – Server-Side Request Forgery (SSRF)
- [ ] Проверка: Не обращается ли сервер к внутренним ресурсам на основе пользовательского ввода?
